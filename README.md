# Sistem Manajemen Taekwondo - Violet Garden

Aplikasi web untuk mengelola data atlet, pelatih, absensi, dan keuangan di klub Taekwondo Violet Garden.

## Fitur Utama

### 1. Manajemen Data
- **Registrasi Data Atlet**: ID, Nama, Tempat Tanggal Lahir, Kelas, Nama Orang Tua, Alamat, No. Telepon, Tingkat Sabuk, No. Registrasi PBTI
- **Registrasi Data Pelatih**: ID, Nama, Nomor Telepon

### 2. Absensi
- **Absensi Kehadiran Atlet**: Nama, Tanggal Latihan, Kehadiran, Keterangan
- **Absensi Kehadiran Pelatih**: Nama, Tanggal, Kehadiran, Keterangan

### 3. Keuangan
- **Iuran Bulanan Atlet**: Nama, Tanggal Bayar, Bulan Iuran, Jumlah, Metode (Transfer/Cash), Keterangan
- **Iuran Kas Bulanan**: Nama, Tanggal Bayar, Bulan Iuran, Jumlah Kas, Metode, Keterangan

### 4. Laporan
- Rekapitulasi Data Atlet
- Rekapitulasi Kehadiran Atlet
- Rekapitulasi Kehadiran Pelatih
- Rekapitulasi Iuran Bulanan
- Rekapitulasi Iuran Kas

## Teknologi
- Frontend: HTML5, CSS3, JavaScript (Bootstrap)
- Backend: PHP/MySQL
- Database: MySQL

## Instalasi
1. Clone repository
2. Setup database
3. Konfigurasi koneksi database
4. Akses aplikasi melalui browser
